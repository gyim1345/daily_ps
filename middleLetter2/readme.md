이해: 
1. s: 문자열
2. return: 중앙 문자
3. 문자열이 짝수이면 2개 반환. 홀수 면 그대로 중앙에 있는 것 반환.

계획:  
1. s 가 짝수이면 중앙 2개 반환.
2. s 가 홀수이면 중앙 1개 반환.

실행:
1. s의 길이를 구함.
2. 짝수 면 중앙에 2개 반환
3. 홀수 이면 1개 반환.
